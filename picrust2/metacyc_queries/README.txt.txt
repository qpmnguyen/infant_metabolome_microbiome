This folder contains results from MetaCyc SmartTable queries. 

There are two types of queries done:
1. Query pathways using list of metabolites  
2. Query metabolites using list of pathways  

The files uploaded to MetaCyc website are metabolite_names.tsv and path_list.tsv  

Result files are PICRUSt2_metabolite_query.csv and PICRUSt2_pathway_query.csv
